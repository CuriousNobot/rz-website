# RZ Website

医疗行业 VC 投资人个人网站静态版。

## 本地预览

直接打开 `index.html` 即可预览；也可以在当前目录启动静态服务：

```bash
python3 -m http.server 8080
```

然后访问：

```text
http://localhost:8080
```

## 可替换内容

- `index.html` 中的 `RZ`、`hello@example.com`、LinkedIn 链接可替换为真实信息。
- 投资方向、交易阶段、内容栏目可按真实履历继续细化。
- 当前版本保留黑白科技感视觉，避免过度营销化，适合作为医疗 VC 的专业入口。
